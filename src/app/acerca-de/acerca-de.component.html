Para el desarrolo de este proyecto se ha usado el framework de CSS llamado Bootstrap para agregar estilos a la aplicación. En el laddo del servidor se ha usado una base de datos MariaDB gestionada a través de XAMPP con el legunaje de programación PHP. Y en el lado del cliente se ha usado el framework Angular y también JQuery que acompaña a Bootstrap a la hora de hacer la funcionalidad de "arcordion" de esta página, por ejemplo.

Se ha usado la directiva *ngIf para en "listar.recetas.component" para advertir al usuario si la base de datos estuviese vacía. En el caso de que esta base de datos contuviese recetas guardadas se ha usado también la directiva *ngFor para listar las recetas.

Por último, se ha usado la directiva *ngStyle para cambiar el color de fondo del componente "header.component".
